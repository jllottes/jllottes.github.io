The Collatz Conjecture¶

Consider a function $f(n)$ defined as follows:

$$ f(n) = \begin{cases} 3n+1 & \text{if $n$ is odd} \\ \frac{n}{2} & \text{if $n$ is even} \end{cases} $$
In [1]:
def f(n):
    if n % 2 == 0:
        return n//2   # use floor division to return an integer
    else:
        return 3*n+1
In [2]:
f(2)
Out[2]:
1
In [3]:
f(4)
Out[3]:
2
In [4]:
f(3)
Out[4]:
10
In [8]:
n = 3
In [19]:
n = f(n)
print(n)

4

We can iteratively apply this function $f$ to any starting integer. We've seen that if we ever reach $1$, then we enter a cycle of $1 \rightarrow 4 \rightarrow 2 \rightarrow 1 \rightarrow \dots$.

Question: Does every starting integer eventually lead to this cycle?

Exercise: Write a function collatz(n) that takes in an integer n and returns a list [n, f(n), f(f(n)), ..., 1].

In [20]:
def collatz(n):
    orbit = [n]
    
    while n != 1:
        n = f(n)
        orbit.append(n)
        
    return orbit

Back to Pythagorean triples:¶

Recall: for two integers $a$, $b$, the greatest common divisor is the largest positive integer that divides both $a$ and $b$.

In [1]:
def gcd(a,b):
    divisors = []
    for d in range(1, min(a,b)+1):
        if a % d == 0 and b % d == 0:
            divisors.append(d)
            
    return max(divisors)

We can more efficiently find Greatest Common Divisors using the Euclidean Algorithm.

  • Start with two integers $a$, $b$.
  • Subtract the smaller number from the larger number and replace the larger with this new number.
  • Repeat this process until one of the numbers is 0.
  • Once we've reached this point, the other number is the greatest common divisor.

Exercise: Write a Python function to implement the Euclidean algorithm to find the greatest common divisor.

We may want to use max() and min() functions to obtain the larger and smaller number respectively.

In [2]:
def Euclidean(a,b):
    larger = max(a,b)
    smaller = min(a,b)
    
    while smaller != 0:
        larger = larger - smaller
        if larger < smaller:
            larger, smaller = smaller, larger  # exchange values if larger < smaller
    return larger
